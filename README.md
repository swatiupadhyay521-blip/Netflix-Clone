# 🎬 Netflix Clone

A responsive **Netflix India Landing Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of the official Netflix homepage, focusing on responsive design, layout, and modern UI.

## 🚀 Live Demo
https://netflix-clone-swati7.vercel.app/



## ✨ Features

- Responsive Netflix-inspired UI
- Hero section with background image
- Navigation bar with Netflix logo
- Language selector
- Sign In button
- Email subscription section
- FAQ section
- Footer with useful links
- Mobile-friendly design

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/swatiupadhyay521-blip/Netflix-Clone.git
```

2. Open the project folder

```bash
cd Netflix-Clone
```

3. Open `index.html` in your browser.
